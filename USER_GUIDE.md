# Kenyan Homes - User Guide

## For Website Visitors

### Browsing the Site
- No login required to browse properties
- View property details and facilities
- Access all pages freely

### Creating an Account
1. Click "Log In" button in navigation
2. Click "Sign Up"
3. Fill in your details
4. Check email for verification link
5. Click verification link
6. You're logged in!

### Booking a Tour
1. Browse properties in "Explore" section
2. Click "Book Tour via WhatsApp"
3. WhatsApp opens with pre-filled message
4. Send message to start booking

### Contacting Us
1. Scroll to "Contact Us" section
2. Fill in name, email, and message
3. Click "Send Message"
4. We'll respond within 24 hours

### Password Reset
1. Click "Log In"
2. Click "Forgot Password?"
3. Enter your email
4. Check email for reset link
5. Click link and set new password

## For Admin Users

### Accessing Admin Panel
1. Go to: http://yourwebsite.com/admin/
2. Login with admin credentials
3. View dashboard

### Managing Users
1. Click "Users" in admin panel
2. View all registered users
3. Edit/deactivate accounts if needed

### Viewing Contact Messages
1. Click "Contact messages"
2. See all messages from contact form
3. Mark as read/replied

### Viewing Tour Bookings
1. Click "Tour bookings"
2. See all booking requests
3. Mark as confirmed when processed

### User Registration Statistics
- Total registered users
- Recent signups
- Active vs inactive accounts

## Troubleshooting

### Can't Log In
- Check email/password spelling
- Use "Forgot Password" if needed
- Ensure email is verified

### Didn't Receive Email
- Check spam/junk folder
- Wait 5 minutes
- Contact admin for manual verification

### WhatsApp Not Opening
- Ensure WhatsApp is installed
- Check phone number in browser
- Try different browser
